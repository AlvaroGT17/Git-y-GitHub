Para poder crear un repositorio en github lo que abra que hacer es en la pagina una vez iniciemos sesión lo que haremos sera pinchar en "Create repository". Le daremos un nombre al proyecto el cual consistirá en el nombre de usuario que lo crea / el nombre que queramos asignarle, el siguiente paso sera decidir si el repositorio sera privado o publico, y por ultimo lo que aremos sera pinchar sobre "create repository" y si todo va bien nos mostrara la información necesaria para crear un nuevo repositorio desde lineas de comando, si queremos subir un repositorio existente o la opción de importar el código de un repositorio existente, entonces copiaremos las lineas de comando que queramos y la pegaremos en el terminal de git.
